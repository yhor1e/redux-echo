# Redux echo

minimal echo sample using `[Redux](http://redux.js.org/)`.
this is not using `[React](https://facebook.github.io/react/)`.

## how to use

1. set up.

```
$ npm i
$ npm run build
```

2. access to src/index.html with web browser.

## npm scripts

### build
```
$ npm run build
$ npm run build:watch
```

### test
```
$ npm test
$ npm run test:watch
```
